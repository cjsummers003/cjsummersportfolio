Train Scheduler

An Inter-Active Train Sheduling Application

Live Demo GitHub

Project description

Train Scheduler will allow the user to see how much time until the next train arrives as well as add a train to the schedule. The Schedule will show the Train, Destination, Frequency, Arrival Time and How long until the next train.

Once the User adds a train it will show the next train closest to the current time. Which the current time will be displayed at the top once a train has been added.
