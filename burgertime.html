It's Burger Time

This game allows the player to try to match their number with the power number

Live Demo GitHub

Project description

This is a restaurant app that lets users input the names of burgers they'd like to eat the devour it.

Whenever a user submits a burger's name, the app will display the burger waiting to be devoured. Each burger in the "Burgers Available to be Eaten" section also has a `Devour it!` button. When the user clicks it, the burger will move to the "Burgers That Have Been Eaten" section.

The app will store every burger in a database, whether devoured or not.

This is a FUll Stack Application
